# License Server

Мини-панель лицензий для обычного виртуального хостинга с PHP.

Что внутри:
- `index.php` — веб-интерфейс генерации и управления ключами
- `api.php?action=validate` — JSON API, которое вызывает магазин
- `config.php.example` — пример настроек
- `data/licenses.json` — файл хранения ключей

Как развернуть:
1. Скопируйте папку `license-server` на хостинг.
2. Переименуйте `config.php.example` в `config.php`.
3. Задайте сложный `panel_password`.
4. Убедитесь, что папка `data/` доступна на запись для PHP.
5. Откройте `index.php`, войдите и создайте ключ.

Как подключить магазин:
1. В админке магазина откройте раздел лицензии.
2. Укажите адрес папки, где лежит `license-server`, например `https://licenses.example.com`.
3. Вставьте сгенерированный ключ и сохраните.

Проверка из магазина идёт в:
- `https://licenses.example.com/api.php?action=validate`
